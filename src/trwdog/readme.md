# trwdog

## Summary

Kill a program that runs too long

## Description

Execute a command with a watchdog timer.

## Usage

    trwdog <arg>+

## Examples

    trwdog make test

## Current version

0.21.13 Add trglob for platform-independent "find" for testing. Add Antlr4ng target, a new Antlr4 tool and TypeScript runtime.
