# Tragl

## Summary

Display a parse tree using Microsoft Automatic Graph Layout

## Description

Read a parse tree from stdin and open a Windows Form that displays the tree.
This tool is part of Trash, Transformations for Antlr Shell.

## Usage

    tragl

## Example

    trparse -i "1+2" | tragl
