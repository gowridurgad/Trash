# trst

## Summary

Print a parse tree in Antlr4 ToStringTree()

## Description

Output tree using the Antlr runtime ToStringTree().

## Usage

    trst

## Examples

    trparse A.g4 | trst

## Current version

0.20.2 Dotnet 7.0 dependency; desc.xml replacement for pom.xml. NB: not all Trash tools supported yet.
