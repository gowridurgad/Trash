# trst

## Summary

Print a parse tree in Antlr4 ToStringTree()

## Description

Output tree using the Antlr runtime ToStringTree().

## Usage

    trst

## Examples

    trparse A.g4 | trst

0.20.22 Fixes to trconvert, trparse, and trtree. NB: not all Trash tools supported yet.
