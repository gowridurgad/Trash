# trtext

## Summary

Print a parse tree with a specific interval

## Description

Reads a tree from stdin and prints the source text. If 'line-number' is
specified, the line number range for the tree is printed.

## Usage

    trtext line-number?

## Examples

    trxgrep //lexerRuleSpec | trtext
